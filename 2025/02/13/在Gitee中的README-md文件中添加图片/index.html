在Gitee中的README.md文件中添加图片

本文最后更新于 2025年2月13日 下午

在开源项目中，良好的文档是吸引用户和贡献者的重要因素之一。README.md 文件通常是项目的第一印象，其中包含项目的描述、使用说明以及其他重要信息。

上传图片

这个图片上传的仓库可以跟项目在同一个仓库里面也可以是一个单独存放文件的仓库

方法一: 直接上传

  • 登录 Gitee，进入您的项目仓库页面。

  • 点击“上传文件”按钮。

  • 将图片文件拖拽到上传区域，或点击选择文件进行上传。

方法二: 使用 Git 命令

如果在本地管理代码，可以通过 Git 命令上传图片：

  • 使用以下命令提交并推送更改 ，也可以使用代码推送工具 比如 SourceTree 或者直接使用 VSCode

获取图片链接

  • 在 Gitee 的文件列表中找到已上传的图片
  • 点击 原始数据，把链接复制下来
