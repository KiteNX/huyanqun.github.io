（Git 推送代码 + nvm / Node / pnpm）

适用人群：Vue / React / uni-app / Node.js 前端或全栈开发者
芯片支持：Intel / Apple Silicon（M1 / M2 / M3

基础原则（非常重要）

在 macOS 上配置开发环境，顺序一定不能乱：

Xcode Command Line Tools → Git → Homebrew → nvm → Node → pnpm

原因：

  • Git / 编译工具 依赖 Xcode
  • nvm 依赖 shell 环境
  • pnpm 依赖 Node

常见问题 & 踩坑总结

❌ git push 报权限问题

  • 没用 SSH
  • Key 没加 ssh-agent

❌ node -v 有，但 pnpm 没有

  • 没执行 pnpm setup
  • PNPM_HOME 没进 PATH

❌ nvm command not found

  • .zshrc 没 source
  • nvm 环境变量写错路径

❌ brew 装了东西但命令不存在

  • PATH 顺序不对
